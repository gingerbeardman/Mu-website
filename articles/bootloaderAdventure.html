Step 2:Copy to a memory card you device supports(SD or Sony Memory Stick) at "/PALM/Launcher"

Step 3:Plug in memory card and run "HWTests"

Step 4:If you dont see "Dump Bootloader" you have an ARM device, and should stop here!

Step 5:Navigate with the up and down keys to "Dump Bootloader" then push button 4 (Button Layout = 1,2 UP/DOWN 3,4)

Step 6:It will look like nothing happened but only push button 4 once, go to the home screen and copy "BOOTLOADER.BIN" to the memory card

Step 7:Take out the memory card and send me "BOOTLOADER_BIN.pdb" along with your device type.
